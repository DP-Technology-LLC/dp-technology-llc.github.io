Party Time Task

About Party Time Task

The Party Time Task is a way to launch a Playable Item on multiple Roku devices on the same network. Once ready, the "host" Roku device controls the Play and Pause of all the other devices.

Task Settings

  • Party Host
    • Select a Roku Device that is on-line to be the host. Once you your Playable Item is loaded on all devices your host can start the viewing on all Roku Devices.

  • Party Goers
    • The list of on-line Roku devices that will watch the same Playable Item as the host. These devices with pause and play when the host pauses and plays.

FAQs

  • Why would I ever run this?
    • Because you can...you could use this if you have kids and adults together but you will watch the same content in different rooms.
