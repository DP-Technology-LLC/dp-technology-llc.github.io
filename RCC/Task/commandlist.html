Command List Task

About Command List Task

The Command List Task is a way to send a list of commands to your Roku Device. In addition to sending valid Roku Commands, you can also provide RCC commands such as Repeat and Wait.

Task Settings

  • Command List
    • A free-text entry for the commands you want to run. You can also use the Command List Builder to ensure you create valid commands by tapping on the button next to the entry.

  • Name, Push Notification upon completion and Start Time
    • See common RCC task properites here.

FAQs

  • What is a valid Roku Command?
    • Every view within RCC that accepts a list of commands, there should be a BUILD button next to the entry. This button will show a Command List Builder. Within Command List Builder you can tap on the familiar buttons you may have seen on a Roku remote, or buttons that create RCC commands.
  • How do I use the Command List Builder and what are RCC commands?
    • You can view information regarding the Command List Builder and the RCC commands here.
