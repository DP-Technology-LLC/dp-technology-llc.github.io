Account Login Task

About Account Login Task

The Account Login Task is a nice way to store literal values for your user name and password across your services. This task will push the literal characters in the User Name box, execute any defined commands, then send the literal characters in the Password box. NOTE: Do not use this task if you are concerned about people on your network performing advanced network snooping.

Task Settings

  • User Name
    • The set of literal characters that get sent first.

  • Command List between User Name and Password
    • After the User Name is sent, you can send a list of Roku commands before it sends the password. This could be used to naviagte out of the User Name field and into the Password field.

  • Password
    • The set of literal characters that get sent last.

FAQs

  • Is it safe?
    • All literal characters within RCC are sent unencrypted so if your private network allowed access to somebody, then if they do a few advanced traffic monitoring tasks, they could see the individual letters sent.
