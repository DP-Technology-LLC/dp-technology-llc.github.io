Keep Within Task

About Keep Within Task

The Keep Within Task is a way users can keep a Roku Device within a certain channel. If launched from the Devices list, RCC will fetch the current channel and relaunch it if the Roku Device leaves. If launched from the Playable Items list, RCC will launch the Playable Item and if the Roku Device leaves RCC will attempt to launch the Playable Item once again.

Task Settings

  • Channel
    • If launched from the Devices, you can select a channel to launch. If no channel is selected RCC will get the current channel of the selected Roku Device and use that.
  • Afterwards
    • After Keep Within is done running, the task will run what ever commands provided here. This can be useful if you want someone to watch Kids and Family channel for 30 minutes and then to send the Roku device to the Home screen.
  • Push Notification if change occurs
    • A check-box option that will create a notification on your device if the task needs to relaunch.
  • Name, Push Notification upon completion and Start Time
    • See common RCC task properites here.
