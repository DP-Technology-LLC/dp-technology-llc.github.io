Running Tasks

About Running Tasks

Running Tasks are tasks that the user started and RCC is currently running. You can view the details where you can update the settings, or you can view the log messages the Running Task has already logged.

Properites of a Running Task

Every property of a running task is dependent on the task it was created from. Properties that are enabled will update the currently running task.

Tasks/Functions

From the Saved Task list there are a number of tasks/functions you do

  • View Details
    • This will load the specific details of a saved task. It will look the same as if you wanted to a create a new one.
  • View Message
    • Running a Saved Task will create a new task to run.
  • Stop Task
    • Running a Saved Task will create a new task to run.
  • Track Channel Data
    • You can find this feature by tapping on the three dots in the upper right portion of the display. Here you can see all of data collected from running the Channel Tracker Task.
