About Main View

The Main View is the starting point for RCC. The Main View is a tabbed view and each tabbed view contains a list of items.

Tabs of Main View

Currently there are seven tabs in the Main View. Each tab, except Maint, has an About tab with a brief summary. Most tabs also have an ad banner near the bottom. ***Warning if you do not see any banners, you may run into issues running tasks. You can create as many records as you need but running a task is dependent on the ad functionality working.
