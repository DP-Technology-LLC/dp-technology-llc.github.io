About the App

Roku Control Center from DP Technology LLC was created to improve a user's experience with their Roku devices. As long as your device is connected to the same private network as a Roku Device this app can communicate with it! This app was created as a passion project and was not developed by Roku, Inc nor anybody associated with Roku, Inc nor any of the Channel Owners within the lineup within the Rokuverse. You can even use this app on older devices, if you can't find this app in your respective store check here(coming soon)! Check in later as we plan on providing more information on the app including guides, tips and tricks! Thank you for stopping by we hope you enjoy the app!

Tasks (and their backstory) within the app

Since we are not associated with Roku or any channels, functionality is dependent on how the Channel developers created their Channels. Currently The Roku Channel supports most if not all of the functionality within the App.

  • Command List task

  • Sleep task
  • Wake task
  • Volume Control task
  • Commerical Counter task
  • Channel Tracker task
  • Party Time task
  • Keep Within task
  • Status Change task
  • Account Login task
  • Save one of the above task to run in the future
  • Play random content within a created Playlist
  • Install a channel
  • Restart OS
  • Navigate to Roku Setting that determines Mobile Control
  • Push Pin
  • Display the many hidden menus
  • Assign Roku Commands to Device Commands

Known Issues

  • When connectivity to devices is weak, tasks may run into issues.
  • On most newer devices, there is an additional battery saver layer that blocks network traffic. Tasks running in background may run but won't be able to connect to Roku devices unless your device is plugged into a power source.
  • Random, hopefully not too often, bugs.
