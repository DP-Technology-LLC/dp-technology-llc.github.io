Roku Control Center from DP Technology LLC Privacy Policy

This privacy policy governs your use of the Roku Control Center from DP Technology LLC (RCC) app that was created by DP Technology LLC. RCC is an app where you can create records of Roku devices you connect to on a private network, records of Playable Items that help launch Roku channels and play the content within and to interface with Roku devices using the Public Application Programming Interfaces (APIs) available on Roku Devices.

What information does RCC obtain and how is it used?

  • Personal Information
    • RCC does not collect Personal Information at any time. Please contact us here if you experience otherwise.
  • User Provided Information
    • RCC stores any information you provide locally to your device. Data records you create help build API calls to your defined Roku Devices. User Provided Information is not transmitted outside of your private network.
  • Roku Device Information
    • When creating Devices within the app, the app will fetch public information available from the device such as, but not limited to, Model Number, Model Name, Channels Installed, etc...
  • Automatically Collected Information
    • When RCC closes unexpectedly, RCC may send debug information to monitor the stability of the App.
  • Does the Application collect precise real time location information of the device?
    • No.
  • Automatic Data Collection and Advertising
    • RCC is an ad based application. DP Technology LLC does not collect or transmit behavior within the app.

Security

All data records can only be accessed by the application.

By default Roku devices allow Mobile control. You can turn this off with the "Control by Mobile Apps" setting. With that setting disabled mobile applications such as RCC will not be able to communicate with the Roku device.
