Link Launcher

About Link Launcher

The Link Launcher allows you to play content directly on your Roku Device from a simple URL. It can also create and run a RCC task from a rokucc.com link (see FAQ). For example if someone wanted you to watch a show on The Roku Channel, they can send you the URL and RCC can launch the content directly to your Roku Device. RCC

FAQs

  • How does the Link Launcher work?
    • If you have a URL of a show that you can watch in a web browser, chances are the URL has a content id used for the show. Some channels use the same content id whether you are watching in a web browser or on your Roku Device. If the content ids match we can parse the id out and use it to directly launch your content on your Roku Device.
  • When I open a URL using RCC, I get a prompt saying "More Info". What does that mean?
    • Most times this means RCC needs to know the Content Type of your content. Your Roku Device needs to know if the id you are sending is a movie, episode, the season id or even the series id. Hopefully the person sending it to you lets you know what it is.
  • What Roku Channels are supported?
    • Below is a list of Roku Channels are supported. There are 10,000+ channels in the Roku-verse, if you know of one that uses the same content ids, see FAQ above, let us know and we will see if we can support it.
    • The Roku Channel
    • Netflix
    • Tubi
    • Peacock TV*
      • Content ids for individual episodes work, as opposed to content ids for a series.
    • YouTube
    • Amazon Prime Video*
    • Hopefully more will be added as more channels get tested.
  • What is a rokucc.com link and why did somebody send it to me instead of a URL for the channel?
    • Rather than sending the URL of the content provider (therokuchannel.roku.com, netflix.com, tubitv.com, etc...), somebody may send you a rokucc.com link. This link contains all the data necessary to start a RCC task. Right now only the Launch Task is supported but more RCC Tasks will be available in future releases.
  • When I receive a rokucc.com link I see an additional button labeled Save Task, is that different than Save Playable Item?
    • Yes. When you tap Save Playable Item, RCC will create a record in your list of Playable Items. If you tap Save Task, RCC will create a Playable Item (if not saved already) and create a record in your list of Saved Tasks. As of right now only Launch Tasks are supported by Link Launcher but all RCC tasks will eventually be shareable.
